Основные этапы разработки программного обеспечения

На главную Назад

7/10

1. Ознакомьтесь с правилами техники безопасности

2. Ознакомьтесь с этапами разработки программного обеспечения

3. Укажите первый этап разработки программного обеспечения

4. Выберите, на какой стадии разработки осуществляется проверка системы в реальных условиях

5. Отметьте, для чего необходим этап анализа требований

6. Выберите, на каком этапе происходит анализ результатов и внесение корректировок в процесс разработки ПО

7. Укажите, какой этап разработки следует за этапом анализа требований

8. Ознакомьтесь со вспомогательными процессами разработки программного обеспечения

9. Укажите, какой из перечисленных процессов относится к обеспечению качества

10. Соотнесите название вспомогательного этапа разработки ПО и его описание
